# Integration

Folder ini berisi hasil penggabungan seluruh pekerjaan tim backend dan frontend.

## Tugas Integration
- Menggabungkan source code backend dan frontend.
- Memastikan seluruh fitur berjalan dengan baik.
- Menguji koneksi database.
- Melakukan pengujian (testing).
- Memperbaiki konflik hasil merge.
- Menyiapkan versi final aplikasi.

## Struktur

api/          -> API dan logika backend
includes/     -> Konfigurasi dan koneksi database
pages/        -> Halaman aplikasi
docs/         -> Dokumentasi
database.sql  -> Database
